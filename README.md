# Predicting Price Moves with News Sentiment

## Project Overview
This project analyzes financial news sentiment and its correlation with stock price movements.

## Structure
- `notebooks/` → Jupyter notebooks for EDA, Technical Indicators, and Correlation
- `src/` → Python modules with helper functions
- `scripts/` → Scripts to run full analysis
- `tests/` → Unit tests
- `requirements.txt` → Required Python packages
- `.github/workflows/` → CI/CD unit tests
